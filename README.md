# Project Hypernova

## Getting Started

### Dependencies

- SDL2
- SDL2_image
- SDL2_mixer
- SDL2_net

### Download

```sh
git clone https://github.com/daviskauffmann/project-hypernova.git
cd project-hypernova
```

### Build

```sh
make
```

### Build & Run

```sh
make run
```

### Cleanup

```sh
make clean
```
